# CyberTasker
Cyber-Tasker Dashboard manajemen tugas berbasis web untuk mengatur alur kerja harian secara efisien. Proyek ini dibangun dengan fokus pada performa dan pengalaman pengguna, menggunakan Vanilla JavaScript dan LocalStorage API untuk penyimpanan data yang cepat dan privat. Fitur utama:  Manajemen tugas intuitif,  Notifikasi deadline otomatis.
