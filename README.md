Cyber-Tasker 🚀
Cyber-Tasker adalah aplikasi manajemen tugas (task management) berbasis web yang dirancang untuk membantu pengguna mengelola tugas koding, kuliah, dan pribadi secara efisien dan intuitif.

🛠️ Teknologi yang Digunakan
HTML5 & CSS3 (Responsive UI/UX)

Vanilla JavaScript (Logic & Interactivity)

LocalStorage API (Data storage)

✨ Fitur Utama
Task Dashboard: Antarmuka bersih untuk melihat, menambah, dan menghapus tugas.

Deadline Notification: Notifikasi otomatis untuk pengingat tugas.

Data Export: Fitur untuk menyimpan data tugas ke format CSV dan PDF.

Dark Mode UI: Antarmuka modern yang nyaman bagi mata.

🚀 Cara Menjalankan Proyek
Aplikasi ini berbasis web client-side sehingga tidak memerlukan konfigurasi server yang rumit:

Pastikan kamu memiliki web browser (Chrome, Edge, atau Firefox).

Buka folder proyek ini.

Cari file index.html.

Klik dua kali (atau drag-and-drop ke browser) file index.html tersebut untuk membuka aplikasi.

💡 Pengembangan Selanjutnya
Saat ini, data tersimpan secara lokal di perangkat pengguna. Rencana pengembangan ke depan:

Migrasi ke Cloud Database untuk sinkronisasi data antar perangkat.

Implementasi sistem User Authentication (login/signup) yang lebih aman.
